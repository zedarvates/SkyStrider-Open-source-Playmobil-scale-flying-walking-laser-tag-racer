# SkyStrider – Roadmap officielle (2025 → 2027)

### Version actuelle – v1.0 « Laser Tag Racer » (déjà dispo)
- 4 moteurs + 4 jambes rétractables araignée  
- Laser tag classe 1 + ballons qui pètent  
- Poids < 190 g, < 35 €, 100 % salon  
- Open-source complet (STL + code + BOM)

### v1.5 – « Tripod Edition » (Q1 2026 – très facile à sortir)
- Passage à 3 jambes seulement (tripode) → encore plus léger (≈ 140 g)  
- Stabilité en marche conservée grâce à un nouveau pas « tripod gait » dans le code  
- Gain : +2 min d’autonomie vol, coût réduit de 6–8 €  
- Option : petite roue ou patin rétractable au bout de chaque jambe pour rouler au sol en mode « turbo sol »

### v2.0 – « Quadrupède Forest Runner » (été 2026)
- Jambes plus longues et articulées (genou + cheville) → vrai mode quadrupède marche/course  
- Fonction « pince de fête foraine » intégrée :  
  - Mini pince/servo à l’avant (comme les machines à peluches)  
  - Objectif du jeu : attraper des petits objets (bonbons, pièces, mini-figurines) pendant la course  
- Mode « treillis / grimpette » :  
  - Crochets ou ventouses magnétiques aux pieds → grimpe sur grillage, rideau, filet de but  
  - Nouveau jeu : « Capture the Flag » vertical dans le jardin ou le gymnase

### v2.5 – « Swarm Pack » (fin 2026)
- Protocole simple de communication entre 4–8 SkyStriders (ESP-NOW ou nRF24)  
- Modes de jeu coopératifs :  
  - Formation volante (leader + suiveurs)  
  - Attaque/défense en équipe avec lasers  
  - Relais de pince (passer un objet d’un drone à l’autre en vol)

### v3.0 – « Outdoor Explorer » (2027)
- Version 250–400 g avec GPS + caméra FPV minuscule  
- Batterie 2S, moteurs plus costauds  
- Toujours jambes rétractables pour atterrir/marcher sur herbe, sable, cailloux  
- Mission type Geocaching : le SkyStrider va poser/récupérer un petit token dans la nature et revient tout seul

### Bonus idées déjà dans les issues
- Kit « conversion hélium » ultra low-cost (14 €) pour les tout-petits  
- Coques interchangeables (Star Wars, Mario, Pokémon, etc.)  
- Mode « disco avec LEDs + musique synchronisée sur plusieurs unités

Toutes les futures versions resteront :
- 100 % open-source  
- imprimables en 3D à la maison  
- < 50 € la version la plus avancée  
- zéro risque légal (lasers classe 1, < 250 g tant que possible)
